One factor that I think is important to consider when discussing the legaility of web scraping is privacy regulations. This is 
protected by privacy laws therefore scarping it without consent would have consquences. Even if the data is publicly visible 
these laws restrict the collection of personal data. If these laws are violated this can result in a fine. Also, with user rights 
they have the right to delete their data, but if it's been scraped this would make it diffcult to manage.

When deciding to web scrape, I think two guidelines would be to minimize the data and respect the access. One should avoid scraping data 
that is personally identafiable. This would include information like phone numbers, emails, and names. Also, one shouldn't scrape beyond 
logins and authentication. This is because it would be considered unauthorized.
